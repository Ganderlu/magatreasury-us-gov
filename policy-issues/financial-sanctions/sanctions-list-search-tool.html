Sanctions List Search Tool

The sanctions list search tool helps users look up names, alternate spellings, and identifiers across OFAC sanctions lists. It is commonly used to investigate possible matches and to document screening decisions.

How to Use It Effectively

  • Search broadly first: try name variants and partial matches.
  • Validate with identifiers: confirm using address, DOB, passport, or other fields.
  • Document outcomes: keep evidence supporting the final determination.
